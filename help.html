Reporting a Bug

If you find a bug, which is honestly quite likley, please do the following.

1. Right click the page and click "inspect". A new page will open up.
2. Click "Console". Doing so will reveal all errors from the code.
3. Send me a screenshot of the console errors, as well as a detailed report of what the bug is and what is required to reproduce it.

Please reach me through the following email:

Technical Information

On the off chance you care or are curious about the inner workings of this project, read here.

All of the backend data for this project is stored on a firebase realtime database. I did this because 1. it's free to start and 2. I didn't feel like writing an entire backend. All of the data, besides passcodes, is stored unencrypted on this database. There are read and write permission systems in place based on firebase auth.
