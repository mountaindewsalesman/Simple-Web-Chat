List of commands

  • /game
    • /game chess (email of other player) - starts a game of chess with another player. Player must be in the current chat.
  • /chat
    • /chat clear - clears all messages in the current chat
    • /chat save - saves the current chat as a file

Why are there commands? How to use them?

Commands are implimented when I feel too lazy or think it woulden't fit to give a function an actual gui.

The commands are structured the same as Minecraft commands. I thought most people might be farmiliar with those. You start off a command by writing /, then the desired command, then the desired sub subcommand, etc.
For example, to send a game of chess, you would type:
/game chess randomperson@gmail.com

I am also planning on adding more commands as needed. There will be more games, and I am thinking about functions to add add people to a chat or to delete a chat or something.
